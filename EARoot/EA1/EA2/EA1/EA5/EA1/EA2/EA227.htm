1. Process Form request is received by Middleware
2. Middleware checks Form Register for form template identification
3. Middleware checks that a base document has been provided
4. Middleware overlays provided information on form
5. Middleware returns newly created form

3a. Middleware retrieves default document (at 4)
4a. Form Overlay application error (at End)

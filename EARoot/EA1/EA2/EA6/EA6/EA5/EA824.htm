An Enterprise Service Bus (ESB) provides a mechanism for connecting applications.  <br /></p><p><br /></p><p>As capabilities become  more advanced, integration components use the Enterprise Service Bus as a common communications channel where services interact with one another. <br /></p><p><br /></p><p>This layer provides a logical data communication session and services to allow reliable communications between inter-connected systems.<br /></p><p><br /></p><p>For instance, the Master Data services system and the Operational Data Store (ODS) are applications that can be connected to each other and other applications using the ESB.  Applications can call the ESB for data services and, without knowledge of the service provider, will have the service request fulfilled.<br /></p>

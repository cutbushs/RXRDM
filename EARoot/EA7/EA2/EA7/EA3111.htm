Notes: The ebmCID element is a universally unique identifier (UUID) used to track the progress of an initial request through all the layers of the SOA Infrastructure. <br /></p><p><br /></p><p>Every new top level invocation of a service should create a new ebmCID, which should then be propagated to each sub level service that it invokes; these in turn will need to propagate the ebmCID to any services that they invoke and so on down the stack.<br /></p>

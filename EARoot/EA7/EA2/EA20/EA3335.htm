Notes: This is used to determine the Agency that controls serialization. This may also be used to indicate system which assigns the serial number (e.g., OPSM - Oracle Pedigree Serial Management system). It should have a value if and only if Item Serial Control Indicator is on. If Item Serial Control Indicator is null (implying that Serial Control Agency must also be null) then it means that the item is not serialized.
